## Installation

In order to install this project, execute (Do-it, Ctrl+D) the following script in the Playground of your Pharo image:

```Smalltalk
Metacello new
  baseline: 'ClassNameAnalyser';
  repository: 'github://NourDjihan/ClassNameAnalyser/src';
  load.
```
